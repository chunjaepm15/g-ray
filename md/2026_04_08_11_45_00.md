# 작업 리포트: REF-L04-11 청크 정교화 및 UI 개선 계획 v2

**일시**: 2026. 04. 08. 11:45

안녕하십니까, 팀장입니다.

보내주신 피드백을 바탕으로, 단순 단어 나열이 아닌 **유의미한 어구 단위(Meaningful Chunks)**로 데이터를 재설계하려 합니다.

## 1. 주요 개선 사항

### 1-1. 청크 로직 전면 재조정
- `The`, `was`, `said` 등 단독으로는 완벽한 한글 뜻을 가지기 어려운 단어들을 유의미한 덩어리(`The judge`, `was impressed`, `what she said`)로 묶겠습니다.
- 해당 단어들을 하나만 선택했을 때는 자동으로 **"[구조 분석 중...]"** 메시지가 나오게 하여, 학생들이 정확한 덩어리를 찾도록 유도하겠습니다.
- `said`가 "것에"라고 나오던 할루시네이션 문제를 `what she said` 전체를 하나의 의미 단위(`그녀가 했던 말에`)로 묶어 해결하겠습니다.

### 1-2. UI 색상 동기화 (스크린샷 기준)
- "투시 확인하기" 버튼의 색상이 드래그 시와 달라 혼란을 주었던 부분을 해결합니다.
- 스크린샷의 `said` 배경색과 동일한 **스카이 블루 계열**로 버튼 색상을 맞춰 일관성을 주겠습니다.

## 2. 작업 기대 효과
- 문법적으로 타당한 덩어리 학습이 가능해집니다.
- 드래그 색상과 버튼 색상이 일치되어 시각적 흐름이 매끄러워집니다.
