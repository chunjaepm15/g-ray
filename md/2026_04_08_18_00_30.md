# CSS (style_v14.css) 변경 사항 상세 보고 (2026-04-08 18:00)

이 문서는 활동 페이지의 문장 시각화 개선을 위해 `style_v14.css` 파일에 적용된 주요 변경 사항을 기록합니다.

## 📝 주요 스타일 변경점

### 1. 활동 페이지 문장 시각화 (`.sentence-word`)
- **전체 형태**: 버튼처럼 보이게 했던 선(border), 배경색, 그림자(box-shadow)를 모두 제거하여 일반 텍스트 문장처럼 보이게 함.
- **폰트 최적화**: 폰트 크기를 `28px`에서 `32px`로 상향.
- **여백 조정**: 단어 사이의 인위적인 간격(margin/padding)을 제거.

### 2. 하이라이트 효과 (`.sentence-word.selected`)
- **배경색**: 부드러운 하늘색 연한 톤(`var(--sky-lt)`)으로 변경.
- **연속성**: `border-radius` 조정을 통해 드래그한 영역이 하나의 마디처럼 이어 보이게 함.
  - `.start`: 시작 부분만 왼쪽 둥글게 처리
  - `.end`: 끝 부분만 오른쪽 둥글게 처리

### 3. 문법 오류 수정 (Standardization)
- `alignItems` ➔ `align-items`
- `lineHeight` ➔ `line-height`
