# 문장투시경 기능 개선 실무 보고서 (2026-04-07) 📑

안녕하세요! 우리 초보 팀원들을 위해 오늘 진행한 `main.js`와 `student.html`의 주요 변경 사항을 정리한 보고서입니다.

## 1. 개요
사용자 편의성 증대와 학습 효과 극대화를 위해 UI 레이아웃을 확장하고, 오답 발생 시의 피드백 시스템을 강화했습니다.

## 2. 상세 변경 사항

### 📐 레이아웃 및 디자인 (전체 크기 상향)
- **컨테이너 확장**: `maxWidth`를 기존보다 약 10~15% 확장하여(최대 1000px) 고해상도 환경에서도 시원한 화면을 제공합니다.
- **가독성 향상**: 문장 폰트 크기를 `22px -> 26px`로 키우고, 여백을 조정하여 쾌적한 학습 환경을 조성했습니다.

### 🧹 UI 간소화
- **난이도 안내 삭제**: 화면의 복잡도를 줄이기 위해 카드 내의 난이도 뱃지 및 관련 안내 버튼들을 제거했습니다.

### 💡 오답 피드백 시스템 (2 Strike Rule)
- **정답 즉시 공개**: 두 번 연속으로 틀릴 경우, 학생이 헤매지 않도록 정답 구역을 자동으로 표시하거나 답안을 입력해줍니다.
- **오답 가이드 추가**: 단순히 틀렸다고만 하지 않고, 해당 단원의 핵심 문법 포인트를 짚어주는 '오답 가이드' 박스를 하단에 노출합니다.
- **학습 대기 시간**: 가이드를 충분히 숙지할 수 있도록 정답 공개 후 3.5초의 대기 시간을 부여한 뒤 다음 문제로 이동합니다.
