# G-ray 프로젝트 변경 내역 리포트

- **일시**: 2026년 4월 7일 14:57
- **업데이트 대상**: `js/pages/teacher_problem.js`

## 💡 주요 변경 사항 (Changes)

교사용 문제 출제 페이지(`teacher_problem.js`)에 세부 커스텀 기능이 새롭게 적용되었습니다. 교사가 각 단원의 예문들을 직접 자유롭게 편성하고, 각 예문에 할당된 학습 로직을 직관적으로 확인하고 제어할 수 있게 되었습니다.

### 1. 단원 예문 추가/제거 커스텀 기능
- **개별 예문 제어 UI**: 단순하게 상위 3개 문장만 보여주던 것에서 벗어나, 교사가 원하는 예문을 `[제거]` 하거나 하단에서 `[+ 추가]` 하여 문제 풀 목록에 자유자재로 담을 수 있습니다.
- 교사가 구성한 예문 선택 상태(`selectedSentences`)는 별도로 저장되어 관리됩니다.

### 2. 문항별 자동 기본 세팅화
- 기본 설정으로 첫 번째 문제는 자동 재배열, 두 번째는 빈칸 채우기, 세 번째는 영작 문제로 세팅되는 등, 교사의 초기 세팅 번거로움을 덜어주는 기본 양식(Default Settings) 규격이 추가되었습니다.

### 3. 문제 유형별 로직(Logic) 실시간 확인용 '미리보기' 패널 추가
- 교사들이 학생들이 풀 문제의 실제 구현 원리를 직접 확인할 수 있는 **"문제 미리보기 및 통제 영역"**이 적용되었습니다.
- **재배열**: 문장 내의 의미 단위(Chunk)들이 실제로 학생들에게 어떻게 쪼개져 보여질지 미리 확인 가능합니다.
- **빈칸 채우기**: 문법(`grammar`) 키워드나 핵심 동사(`Verb`)에 기반해서 자동으로 Regex(정규식)를 씌워 화면상에 어떻게 `_____` 블라인드 처리되는지 시뮬레이션 합니다.
- **영작**: 제공할 한글 문제와, 채점의 기준이 될 실제 영문 정답이 대조되어 나타납니다.
