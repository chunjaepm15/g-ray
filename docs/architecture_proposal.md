# 🏗️ 문장투시경 (G-ray) v13 아키텍처 제안서

## 🧩 시스템 개요
본 시스템은 중등 영어 교과서의 구문 구조와 핵심 문법을 학습하기 위해 설계된 React 기반의 교육 플랫폼입니다.

## 🛠️ 핵심 로직 (Engine Level)
### 1. **CoT 기반 구문 인식 (Chain of Thought)**
- **본동사 탐색**: 문장에서 핵심적인 동사를 먼저 찾아내어 주어와 서술부의 경계를 확정합니다.
- **절 경계 탐색**: 'what', 'that', 'if', 'whether'와 같은 접속사/관계사절을 전체 덩어리(Clause-Chunk)로 인식하여 수동 분석의 오류를 최소화합니다.
- **EBS 표준 4대 청크**: [S]주어 + [V]동사 + [O/C]목적어/보어 + [M]수식어.

### 2. **AI 데이터 정규화 (scripts/data_rebuild_tool.py)**
- **교과서 원문 파싱**: 원문 텍스트(.json/raw)를 불러와 공통 10대 문법(CG01-CG10)에 맞춰 매핑합니다.
- **통합 빌드**: 이재영/정사열 등 상이한 판권의 데이터를 통일된 AIDT 기술 스펙으로 변환하여 `js/curriculum_v13.js`를 생성합니다.

## 💻 프론트엔드 설계
- **무겁지 않은 로딩**: React 18 & Babel Standalone (UMD) 방식을 채택하여 별도의 빌드 과정 없이 정적 HTML 파일만으로 AIDT 기능을 완벽하게 구동합니다.
- **역할 기반 뷰 (Role-Based View)**:
    - **Teacher View**: 실시간 학급 대시보드 및 지문 분석 기능 제공.
    - **Student View**: 문장 분석, 빈칸 완성, 영작 등 3단계 반복 훈련 시스템.
