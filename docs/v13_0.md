### 📊 2. 커리큘럼 데이터의 고도화 (v13.1.1)
- **40개 정예 예문 적용**: 10개 핵심 문법 유니별 4개씩, 총 40개의 엄선된 문장 데이터셋을 새롭게 추출하여 반영했습니다.
- **구문 분석 정밀화 (S/V/O/C/M)**: 
    - 기존에 뭉뚱그려져 있던 `Total` 역할을 주어, 동사, 목적어, 보어, 수식어 단위로 완벽하게 분할했습니다.
    - 한국어 라벨을 **"주어(S)"**, **"동사(V)"** 등 표준형식으로 통일하여 학습 직관성을 높였습니다.
- **고정밀(REF) 엔진 활성화**: 모든 문장 ID 앞에 `REF-`를 부여하여, 앱이 자체 계산이 아닌 파일 내의 정밀 분석 데이터를 사용하도록 강제했습니다.
- **AI 피드백 복구**: `if/whether` 및 `too~to` 구문 등 주요 패턴에 대한 `variation_flags`를 재설정하여 학습 지원 힌트 기능을 정상화했습니다.

### 🎨 3. UI/UX 및 기능 개선
- **동적 렌더링 도입**: 하드코딩된 SVG 구문을 제거하고, `syntax_chunks` 데이터를 기반으로 한 **동적 화살표 및 컬러 박스 렌더링**을 구현했습니다.
- **아코디언 인터페이스**: 문장 분석 시 필요한 정보만 선택해서 볼 수 있도록 [문장 구조 투시], [개념 정리] 섹션을 기본적으로 접힘 상태로 최적화했습니다.
- **Voca 자동 추출**: 분석 중인 문장에서 주요 단어를 AI가 동적으로 추출하여 표시하는 기능을 적용했습니다.
- **교사용 관리 도구**: 단원별 문제 유형(빈칸, 재배열 등)을 설정할 수 있는 고도화된 모달 레이아웃을 도입했습니다.
