# [기안] 중학 3학년 영어 문법 AIDT 학습 서비스 기획 및 데이터 구조 정의서

## 1. 개요 (Overview)
본 문서는 천재교육 AIDT(AI Digital Textbook) 프로그램의 일환으로 개발되는 **중학 3학년 영어 문법 학습 서비스**를 위한 데이터 기획 및 RAG(Retrieval-Augmented Generation) 지식베이스 설계를 목적으로 합니다. 

### 1-1. 서비스 핵심 목표
- **문장 투시경 기술 적용**: 구문독해, 스캐닝, 직독직해, 끊어읽기 과정을 통해 자연스러운 문장 독해력 함양.
- **AI 맞춤형 가이드**: RAG 기반의 지식 시스템을 활용하여 학생의 개별 질문에 교과서/문제집 기반의 정확한 답변 제공.
- **교사/학습 유연성**: 매년 변동되는 단원 구성 및 교수법에 유연하게 대응 가능한 데이터 구조 확보.

---

## 2. 서비스 아키텍처 및 데이터 활용 전략
### 2-1. 지능형 답변 프로세스 (Data Hierarchy)
1. **1순위 (RAG 기반)**: 구축된 `데이터 문서(JSON/DB)`에서 해당 단원의 핵심 문법, 예문, 설명 데이터를 검색하여 답변 생성. (정확도 확보)
2. **2순위 (LLM 폴백)**: 매칭되는 기존 데이터가 없을 경우, 사전 학습된 LLM이 교육과정 가이드라인 내에서 생성형 답변 제공.

### 2-2. 단원의 구성과 상세 설명 (Template)
각 단원은 단순히 문법 항목만 나열하는 것이 아니라, 다음의 다차원 데이터를 포함합니다.

| 항목 | 상세 내용 |
| :--- | :--- |
| **주요 문법** | 해당 단원의 핵심 타겟 문법 (예: 관계대명사 what, 과거완료 등) |
| **구문 형식** | 문법이 쓰이는 기본 뼈대 문장 구조 (S+V+O...+등) |
| **지문 활용** | 교과서 본문 내에서 해당 문법이 실제 독해에 어떻게 기여하는지 분석 |
| **스피킹 예시** | 일상 대화 상황에서 해당 구문을 사용하는 구어체 예문 |
| **상황별 사용법** | 공식적인 글쓰기 vs 비공식적인 대화에서의 차이점 |
| **외부 참조** | 유튜브(밀크T, EBS), 네이버 블로그 강의 링크 연계 |
| **출처 정보** | 천재교육 교과서(소영순/이상기), 문제집(100발100중 등) 페이지 정보 |

## 3. 효과적인 교육법 및 교수 전략
중학 3학년 학생들이 특히 어려워하는 부분에 대한 AIDT 솔루션을 제안합니다.

### 3-1. 학습 고충(Pain Points) 분석 및 해결책
1. **과거완료 vs 현재완료의 혼동**
   - **이유**: 한국어에는 없는 시계열적 선후 관계 개념이 낯설음.
   - **해결책**: '시간의 선형 그래프 UI'를 통해 사건 A와 사건 B의 발생 순서를 시각적으로 선사함.
2. **관계대명사 what의 격 변화**
   - **이유**: 선행사를 포함한다는 개념과 '것'으로 해석되는 명사절 성격을 파악하기 어려움.
   - **해결책**: 'The thing which'라는 분해형 문장과 'What' 문장을 나란히 배치하여 투시경 모드로 구조적 동일성 확인.

### 3-2. 전공 교사 커리큘럼 반영
- **스킬**: '끊어읽기(Chunking)' 단위를 수준별로 조정 (기초: 단어 위주 / 심화: 구·절 위주).
- **도움 커리큘럼**: 단원 시작 전 '이전 학년 기초 문법 복습' 세션을 5분 내외로 배치하여 학습 단절 방지.

## 4. 학습 컨텐츠 리소스 가이드 (중3 천재교육 특화)
- **유튜브**: "중3 천재 소영순 [단원명]", "중3 천재 이상기 내신대비" 키워드 기반 강의 자동 큐레이션.
- **블로그**: 네이버 '영어 교육' 전문 블로거들의 요약 노트 및 예상 문제 자료 RAG 인덱싱.
