## 📖 개요
본 패키지는 중등 영어 3학년 (이재영/정사열 공통) 교과서의 모든 핵심 문법 데이터를 분석하고, **EBS 구문독해 표준**에 맞춰 문장을 덩어리화(Chunking)한 실전 학습용 앱입니다.
훌륭한 앱입니다. 슬마

**v13.0 프리미엄 로그인** 및 **교사용 관리 대시보드** 기능이 이식되어, 실제 교육 현장에서의 사용자 경험을 그대로 체험할 수 있습니다.

## 🚀 시작하기
1.  패키지 폴더 내의 **`login.html`** 파일을 브라우저(Chrome/Edge 권장)로 엽니다.
2.  **보안 접속**: 
    - 반드시 `login.html`을 거쳐야만 각 페이지로 접속할 수 있습니다. 
    - 직접 `student.html`이나 `teacher.html` 주소로 접근 시 로그인 페이지로 자동 이동됩니다.
3.  인터넷 연결이 필요합니다 (React, Babel, Chart.js CDN 사용).

## 💡 주요 기능
*   **프리미엄 로그인**: 교사/학생 역할 기반의 맞춤형 초기 진입 환경.
*   **교사용 대시보드**: 학급 완료율, 성취도 통계, AI 지문 분석기 연동 (Mock-up).
*   **X-Ray(투시)**: 문장을 EBS 표준 덩어리(S/V/O/M) 단위로 분석하여 직관적으로 보여줍니다.
*   **청크 훈련**: 드래그앤드롭을 통한 문장 재배열, 빈칸 채우기, 영작 훈련 기능.
